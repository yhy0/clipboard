# Release Notes

## 🎉 0.3.0 (2026-01-01)

### What's new
- 提供按类型、应用或日期更丰富的过滤方式，以便于更好的查询历史记录
- 关键字搜索支持高亮展示
- 文本支持编辑并保存
- 支持暂停并恢复剪贴板记录

### Improvements
- 改进链接、图片卡片展示
- 预览窗口体验优化
- 设置-关于界面的图标支持直接拖拽获取权限
- 更好的检测应用更新
- 兼容macOS Sonoma

### Bug Fixes
- 修复了一些已知问题

## 🎉 0.2.2 (2025-11-21)
